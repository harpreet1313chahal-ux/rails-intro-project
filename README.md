# Rails Intro Project

A Ruby on Rails Book Library application developed for the Rails Intro Project.

## Features

- Browse books
- Browse authors
- Browse categories
- Search books
- Filter by category
- Sort books
- Pagination
- Bootstrap styling
- About page
- Dataset page

## Technologies

- Ruby 3.4
- Rails 8
- SQLite3
- Bootstrap
- Pagy
- CSV

## Setup

```bash
bundle install
rails db:migrate
rails db:seed
rails server
```

Open:

http://localhost:3000
